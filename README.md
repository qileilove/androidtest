##<intent-filter>标签中配置一个<data>标签
    1. android:scheme 
    用于指定数据的协议部分,如上例中的 http 部分。
    2. android:host
    用于指定数据的主机名部分,如上例中的 www.baidu.com 部分。
    3. android:port
    用于指定数据的端口部分,一般紧随在主机名之后。
    4. android:path
    用于指定主机名和端口之后的部分,如一段网址中跟在域名之后的内容。
    5. android:mimeType
    用于指定可以处理的数据类型,允许使用通配符的方式进行指定。
